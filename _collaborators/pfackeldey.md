### Biography

Peter Fackeldey is a Postdoctoral Research Associate at Princeton University. He develops software for high-energy physics, maintaining the Awkward Array and dask-awkward projects. He has been a CMS collaboration member since 2016, focusing on Higgs analysis with Run 2 data.

### Project interests

Peter's technical work focuses on high-performance Python software for numerical computation, complex data structures (particularly ragged and nested arrays), and statistical analysis tools for particle physics research. He is investigating ways to efficiently work with arrays, including JIT compilation for array-computing (JAX, Numba, warp), and exploring which areas from other scientific fields have similar computational challenges to HEP and whether common solutions can be found.
